Servicio API RESTful de geocodificación de direcciones del Sistema Cartográfico Nacional de España


El Servicio API de geocodificación de direcciones del Sistema Cartográfico Nacional de España, tiene como objetivo buscar y obtener la localización geográfica e información de cualquier dirección que esté ubicada dentro del territorio español, a partir del análisis de los datos identificativos suministrados por los usuarios.

La URL PROVISIONAL DEL SERVICIO (EN FASE DE PRUEBAS) es: https://geocoder-5-ign.larioja.org/v1

Además de la geolocalización de una dirección específica, este servicio permite también recabar los datos de las direcciones cercanas a un punto determinado, así como aquellas que se encuentren comprendidas dentro de un área rectangular o un círculo dado.

El servicio utiliza internamente la API Pelias con objeto de ofrecer un conjunto de funciones de consulta flexibles y sofisticadas.

Los métodos (endpoints) y parámetros que se describen en este documento, permiten a cualquier usuario identificar y localizar direcciones con la máxima precisión a partir de los datos completos o parciales que definen una dirección, de su identificador (ID) o de un nombre representativo de la misma.

Las búsquedas interrogan al conjunto de datos de direcciones que se encuentra almacenado en una base de datos ElasticSearch (ELS), siendo el binomio Elasticsearch-API Pelias, junto con distintos procedimientos desarrollados ad-hoc para este servicio, los elementos que conforman la plataforma tecnológica que provee la funcionalidad buscada.

Los datos básicos de las direcciones de referencia son recopilados a partir de la información proporcionada por los diferentes productores con competencias en materia de direcciones, dentro del marco de cooperación del Sistema Cartográfico Nacional de España.

Funcionalidad

El servicio permite consultar la API RESTful con el fin de:

  • Obtener los datos y la ubicación precisa de una dirección determinada.
  • Localizar la dirección más próxima a un punto determinado.
  • Localizar las direcciones que se encuentran dentro de un área rectangular o circular.
  • Realizar búsquedas selectivas entre el conjunto de direcciones de un productor dado.

La API de direcciones Pelias

La API de direcciones Pelias utilizada en este servicio, es una API RESTful que devuelve en formato JSON el resultado de las consultas realizadas conforme a los métodos y parámetros definidos en el presente documento.
